# 📚 Study Countdown Timer

A simple, customizable countdown timer built with **HTML, CSS, and JavaScript**.  
Designed specifically for study sessions, different from Pomodoro timers — this lets you set **any custom interval**.

---

## ✨ Features
- ⏱️ Input fields for hours, minutes, and seconds
- ▶️ Start, ⏸️ Pause, 🔄 Reset controls
- 🔔 Sound alert when time is up
- 🖥️ Fullscreen mode option
- 🎨 Clean, responsive design

---

## 🚀 How to Use
1. Enter your desired time (hours, minutes, seconds).
2. Click **Start** to begin the countdown.
3. Use **Pause** to stop temporarily.
4. Use **Reset** to clear and restart.
5. When the timer ends, a sound alert will play.
6. Click **Fullscreen** for distraction-free study mode.

---

## 🛠️ Tech Stack
- **HTML** → Structure
- **CSS** → Styling
- **JavaScript** → Timer logic

---

## 🎯 Example
- Set `00:25:00` → 25-minute study session.
- Timer counts down and plays alert when finished.
- Perfect for flexible study intervals.
