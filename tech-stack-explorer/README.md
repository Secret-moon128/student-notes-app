# Tech Stack Explorer with Learning Paths & Technology Comparison Dashboard 🚀📊

An interactive platform to help developers explore popular tech stacks, follow structured learning paths, and make informed choices with a technology comparison scorecard.

## 🌟 Features

1. **Interactive Tech Stack Selector**:
   - Dive into MERN, MEAN, LAMP, or Jamstack. View features, setup difficulty, and use cases for every component.
2. **Interactive Learning Paths**:
   - Check off roadmap steps with dynamic progress bar tracking.
3. **Side-by-Side Tech Comparison**:
   - Choose two technologies (e.g. React vs Angular, Node.js vs Python) and compare them across ratings (Performance, Scalability, Community, Learning Curve) and pros/cons.
4. **Custom Stack Learning Goals (CRUD)**:
   - Log custom technology learning items, dates, and track target completions.

## 🛠 Setup

To run locally, start a server or open `index.html` directly in your browser:
```bash
npm run dev
```
Navigate to `/tech-stack-explorer/index.html`.
