# 🔍 Regex Tester Playground

A modern **Regex Tester App** built with **HTML + CSS + JavaScript**.  
CS students can test regex patterns, highlight matches in real-time, and view match counts and groups.

---

## ✨ Features
- 🖊️ Input regex pattern and test string
- 👀 Real-time highlighting of matches
- 🔢 Match count display
- 📂 Group matches shown in list
- 🎨 Clean card-style responsive UI

---

## 🖥️ Demo
Open `index.html` in your browser to start testing regex instantly.
