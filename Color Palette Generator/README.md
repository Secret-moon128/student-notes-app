# 🎨 Color Palette Generator

A modern **Color Palette Generator** built with **HTML + CSS + JavaScript**.  
Generate random palettes, lock colors, and copy hex codes instantly.

---

## ✨ Features
- 🔄 Generate 5 random colors
- 🔒 Lock individual colors to keep them
- 📋 Copy hex codes on click
- 🎨 Clean card-style responsive UI
- ⚡ Smooth hover animations

---


## 🖥️ Demo
Open `index.html` in your browser to start generating palettes instantly.
