# ⌨️ Typing Speed Test App

A professional **Typing Speed Test App** built with HTML, CSS, and JavaScript.  
Helps students measure typing speed (WPM) and accuracy with random paragraphs.

---

## ✨ Features
- Random motivational paragraph each test
- Tracks Words Per Minute (WPM)
- Tracks accuracy percentage
- 60-second timer
- Shows final results after test ends
- Clean, responsive UI

---

## 🚀 How to Use
1. Click **Start Test**.
2. Type the displayed paragraph in the input box.
3. Watch live stats update (WPM + Accuracy).
4. After 60 seconds, see your final results.
5. Click **Reset** to try again.
