# Startup Resource Hub 🚀💡

A complete web platform built to help founders launch successful businesses. Leverage interactive planning frameworks, financial projection sheets, slide deck outlines, curated articles, and checklist tools.

## 🌟 Features

1. **Startup Checklist (CRUD)**:
   - Dynamic launch checklist mapping progress indicators (progress rings).
2. **Business Plan Builder**:
   - Write and persist essential business pillars (Executive Summary, Market Analysis, Financial Strategies).
3. **Break-even Financial Calculator**:
   - Project profit margins and units sold needed to cover overhead costs.
4. **Pitch Deck Outliner**:
   - Slide template generator that exports summaries.
5. **Guides & Material Links**:
   - Organized tabs (Ideation, Scaling, Marketing) with read checkmarks.

## 🛠 Setup

To run locally, start a server or open `index.html` in your browser:
```bash
npm run dev
```
Navigate to `/startup-resource-hub/index.html`.
