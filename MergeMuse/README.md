# 🌸 MergeBloom

MergeBloom is a beautiful GitHub analytics dashboard that helps users explore GitHub profiles, repository statistics, language usage, and developer insights through an elegant pink-themed interface.

## ✨ Features

- 🔍 Search any GitHub username
- 👤 View profile information
- 📊 Analyze repository statistics
- ⭐ Track stars and forks
- 💻 Visualize programming languages
- 📈 Interactive charts and analytics
- 🎀 Modern pink & white UI
- 📱 Fully responsive design

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub REST API
- Chart.js

## ⚙️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/MergeBloom.git
```

2. Open the project folder

```bash
cd MergeBloom
```

3. Run `index.html` in your browser or use Live Server.
