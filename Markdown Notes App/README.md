# 📝 Markdown Notes App

A modern **Notes App** with **Markdown support** built using **HTML + CSS + JavaScript**.  
Write notes in Markdown, see a live preview, and save them with localStorage persistence.

---

## ✨ Features
- 🖊️ Write notes in Markdown
- 👀 Live preview panel
- 💾 Save notes to localStorage
- 📂 View saved notes with rendered Markdown
- 🎨 Clean card-style responsive UI
