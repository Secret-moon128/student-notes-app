# 🎓 Student Budget Planner

A simple and attractive **Budget Planner** built with HTML, CSS, and JavaScript.  
Helps students plan monthly budgets, track planned vs actual spending, and visualize remaining balance.

---

## ✨ Features
- Set a monthly budget
- Add categories with planned vs actual spending
- Automatic calculation of differences
- Remaining balance with color indicators:
  - 🟢 Green → Healthy balance
  - 🟠 Orange → Moderate balance
  - 🔴 Red → Low balance
- Clean, responsive UI

---

## 🚀 How to Use
1. Enter your monthly budget.
2. Add categories (Food, Rent, Books, etc.) with planned and actual spending.
3. See differences and remaining balance update automatically.
4. Balance color shows financial health.
