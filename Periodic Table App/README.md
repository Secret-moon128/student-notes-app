# 🧪 Periodic Table Interactive App

An interactive **Periodic Table App** built with **HTML + CSS + JavaScript**.  
Chemistry students can explore all 118 elements, click to view details, filter by category, and search quickly.

---

## ✨ Features
- 📊 Display all 118 elements in grid layout
- 🖱️ Click element to view details (atomic number, mass, category)
- 🗂️ Filter by category (Metals, Nonmetals, Metalloids)
- 🔍 Quick search by name or symbol
- 🎨 Clean card-style responsive UI
